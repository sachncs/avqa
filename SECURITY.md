# Security

If you discover a security vulnerability in AVQA, please report it
privately via the GitHub issue tracker's private reporting channel.
Do **not** open a public issue for suspected vulnerabilities.

We aim to acknowledge reports within 72 hours and to publish a fix
within 30 days for high-severity issues.

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

Earlier versions are not supported.
